------------------------------------------------------------------------------------------------------------------------------

功能介紹
----------------------------------------------------------------------------------------------------------------
### 前台功能 : 
1. 載入頁面動畫
2. 會員註冊
3. 會員登入登出
4. 商品列表頁
5. 商品資訊頁
6. 商品追蹤
7. 輪播圖
8. 客製化團服
9. 加入購物車
10. 收藏商品
11. 訂單資訊
12. RWD響應式設計
